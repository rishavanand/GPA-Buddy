Install this App

GPA Buddy can now be installed as an app by the power of PWAs.

For Android
    1. Open Chrome Browser
    2. Goto gpa.rishavanand.com
    3. Tap on icon.
    4. Choose Add to Home Screen
    5. Wait for it to install
For iOS
    1. Open Safari Browser
    2. Goto gpa.rishavanand.com
    3. Tap on icon.
    4. Choose Add to Home Screen
    5. Wait for it to install
